Getting Started

Welcome to the FlowPay API. This API enables you to:

  • Create and manage wallets
  • Fund and withdraw from wallets
  • Initiate and track payments
  • Retrieve transaction history
  • Fetch FX rates for supported currencies

All requests require a valid JWT token for protected routes. Idempotency keys are supported for safe retrying of financial requests.

View API Documentation

Authentication

Endpoints for user registration, login, and profile management:

POST /api/auth/register POST /api/auth/login GET /api/auth/me

Wallets & Payments

Endpoints for managing wallets and payments:

POST /api/wallets/fund POST /api/wallets/withdraw POST /api/payments

For full API details, visit the OpenAPI/Swagger documentation.
